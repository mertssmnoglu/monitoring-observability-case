# ClickHouse

## Environment Variables

```shell
CLICKHOUSE_PASSWORD=CHANGE_WITH_YOUR_SUPER_SECRET_PASSWORD_PLEASE
```

To set up the environment variables, copy the example file:

```shell
cp .env.example .env
```
